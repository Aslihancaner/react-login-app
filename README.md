React + MUI login starter

Nasıl çalıştırılır:

1) Node.js >= 18 yüklü olmalı.
2) Terminalde proje dizinine gelin:
   cd react-login-app
3) Paketleri yükleyin:
   npm install
4) Geliştirme modunda çalıştırın:
   npm run dev

Frontend, backend olarak http://localhost:8000 adresindeki FastAPI login endpoint'ine istek atacak şekilde ayarlandı.
